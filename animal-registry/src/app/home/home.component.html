A través de esta aplicación web podrá:

  • Crear un animal
  • Editar un animal
  • Eliminar un animal
  • Listar todos los animales del Zoologico

    • ¿Cómo usar la aplicación?


    En la parte superior encontrara una barra de navegacion con tres items:

    • Animals:

      • El boton 'Animals' le permitira ver la lista de los animales registrados en nuestra base de datos.

    • Register:

      • El boton 'Register' le permitira registrar un nuevo animal.

    • Search:

      • El boton 'Search' le permitira buscar y editar un animal que este registrado en la base de datos, para ello debe escribir en el campo el ID del animal.

    Condiciones:

    La especie seleccionada para este api es el puma.

    • Los nombres de los animales deben ser unicos y no pueden superar los 120 digitos de longitud y solo pueden contener letras y espacios.
    • La fecha de llegada no puede ser posterior a la fecha actual pues solo se lleva registro de los animales que estan actualmente en el zoologico.
